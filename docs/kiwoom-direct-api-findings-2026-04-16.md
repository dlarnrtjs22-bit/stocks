# 키움 직접응답 TR 탐색 결과

작성일: 2026-04-16

## 사용자 앱 기준 목표값

- 현대차 평가손익: `+763`
- 두산에너빌리티 평가손익: `+6,578`

## 실제 호출 결과 요약

### 1. `kt00018` 계좌평가잔고내역요청

- 종목별 손익:
  - 현대차: `-4,228`
  - 두산에너빌리티: `-1,805`
- 총 평가손익:
  - `-6,033`
- 결론:
  - 사용자 앱 값과 불일치
  - 장 종료 후 앱 숫자와 직접 매칭되는 TR로 보기 어렵다

### 2. `kt00005` 체결잔고요청

- 종목별 손익:
  - 현대차: `-4,228`
  - 두산에너빌리티: `-1,805`
- 종목별 평가금액:
  - 현대차: `532,772`
  - 두산에너빌리티: `442,195`
- 총 평가손익:
  - `-6,033`
- 결론:
  - `kt00018`과 같은 방향의 값
  - 사용자 앱 숫자와 불일치

### 3. `ka01690` 일별잔고수익률

- 종목별 손익:
  - 현대차: `-4,228`
  - 두산에너빌리티: `-1,805`
- 총 평가손익:
  - `-6,033`
- 결론:
  - 정규장 기준 계좌평가 계열과 동일
  - 사용자 앱 숫자와 불일치

### 4. `ka10085` 계좌수익률요청

- 종목별 현재가:
  - 현대차: `539,000`
  - 두산에너빌리티: `112,900`
- 종목별 매입단가:
  - 현대차: `537,000`
  - 두산에너빌리티: `111,000`
- 직접 손익 필드:
  - 없음
- 결론:
  - 사용자 앱 화면과 같은 `장후 시세`를 주는 유력 후보
  - 하지만 앱과 동일한 `평가손익` 숫자를 직접 주지는 않는다

### 5. `ka10074` 일자별실현손익요청

- 기간 일자별 실현손익 요약 조회 가능
- 결론:
  - 매매내역 탭의 `일자별 손익`에는 적합
  - 보유종목 실시간 평가손익과는 직접 관계가 약함

### 6. `ka10076` 체결요청

- 오늘 실제 체결내역 조회 가능
- 확인된 체결:
  - 현대차
  - 두산에너빌리티
- 결론:
  - 매매내역 탭의 `실제 체결내역`에는 적합
  - 보유종목 실시간 평가손익과는 직접 관계가 약함

### 7. `kt00003` 추정자산조회요청

- `prsm_dpst_aset_amt` 같은 추정자산 계열 값 확인 가능
- 결론:
  - 추정자산 참고용
  - 종목별 평가손익 직접응답 아님

## 중간 결론

### 확정된 사실

- 정규장 기준 손익/평가를 직접 주는 TR:
  - `kt00018`
  - `kt00005`
  - `ka01690`
- 장후 시세를 직접 주는 유력 TR:
  - `ka10085`

## 최종 결론

- 사용자 앱과 동일한 장후 평가손익/평가금액은
  `kt00018` 또는 `kt00004`에
  `dmst_stex_tp = NXT`
  로 호출했을 때 직접응답으로 내려온다.

### 실제 일치 확인값

- 현대차
  - 평가손익: `+763`
- 두산에너빌리티
  - 평가손익: `+6,578`
- 총 평가금액: `990,600`
- 총 평가손익: `7,341`
- 추정자산: `1,007,341`

### 적용 기준

- 정규장 중:
  - `dmst_stex_tp = KRX`
- 장 종료 후 / NXT 시간대:
  - `dmst_stex_tp = NXT`

### 구현 결정

- `매매내역` 탭의 계좌 요약/보유종목은
  `kt00018` 직접응답을 그대로 사용한다.
- 체결내역은 `ka10076`
- 기간 손익은 `ka10074`

### 아직 미확정인 부분

- 사용자 앱의 `+763`, `+6,578` 같은 장후 평가손익을
  완전히 동일한 숫자로 직접 내려주는 TR은 아직 못 찾았다.
